An exercise in using JavaScript objects

This rotating widget that ran on Freep.com's Politics category page for most of 2008 offered me a chance to create a class of JavaScript objects and coordinate their shared properties.

I know, I know... Technically, JavaScript doesn't have classes.

Still, each candidate has several properties in common with the others. They all have an image with identical dimensions that appears when they're on display. Those images all need to be pre-cached, and they all need to declare the path where my script can find them. They are all either left- or right-aligned. All had a destination: a page full of in-depth coverage that readers could see by clicking on their names.

My job was to swap these distinct values with new ones every 5 seconds.
